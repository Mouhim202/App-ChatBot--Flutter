## 🔐 Setup & Configuration

1. Clone the repo:
   ```bash
   git https://github.com/Mouhim202/App-ChatBot--Flutter.git
   cd chatbot-flutter
   ```

2. Add your `.env` file:
   ```
   OPENAI_API_KEY=your_openai_key_here
   ```

3. Install dependencies:
   ```bash
   flutter pub get
   ```

4. Run the app:
   ```bash
   flutter run
   ```
